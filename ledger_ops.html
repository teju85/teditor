All operations that are supported in ledger-mode.

ledger-open

Opens the ledger file in a new Buffer, if not already done, and switches to it. Location of this file can be configured with the Option ledger:file.

Note
Available since v1.3.0

ledger-top

Shows all top-level account transaction summary inside a *ledger Buffer.
